# Система модерации объявлений (тестовое задание Avito, Frontend осень 2025)

Веб-приложение для модерации объявлений: список, детальная страница и статистика модератора.

## Стек

- Node.js 20
- Vite + React 18
- TypeScript
- React Router 6
- @tanstack/react-query — работа с HTTP, кеширование, отмена запросов
- Axios — HTTP-клиент
- Без UI-китов (верстка на чистом CSS)

### Почему такие технологии

- **Vite + React + TS** — стандартная связка, быстрый dev-сервер и типобезопасность.
- **React Query**:
  - кеширует ответы,
  - автоматом отменяет запросы при размонтировании компонента (прерывание запросов при переходе по страницам),
  - упрощает обработку состояний загрузки/ошибок.
- **Axios** — единый клиент с базовым URL `/api`, удобно подключать интерсепторы и типизацию.

## Запуск

```bash
# установка зависимостей
npm install

# dev-режим
npm run dev

# production сборка
npm run build
npm run preview
```
